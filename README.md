## Getting Started

1. Clone this repository or use this template
2. Change the content of `site.config.cjs` to your own
3. Add `NOTION_TOKEN`, `NOTION_DATABASE_ID`, and `NOTION_FEED_ID` to your Vercel environment variables
4. Deploy to Vercel

That's it!
